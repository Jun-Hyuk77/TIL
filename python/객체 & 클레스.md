## object(객체) & class 

1. object(객체)란?

   - 속성과 행위로 구성된 대상.

   - 변수와 함수의 묶음

   

2. class 선언

   1. 객체를 만들기 위해서는 클래스를 선언해야한다.  
      -  `class '키워드' ():` 키워드는 영대문자를 사용.
   2.  class 내 변수를 설정하고 `def 함수():` 형태로 함수 작성



3. object 초기화 (__init__)
   - `__init__()` : 객체 생성 및 속성값 지정 가능
   - init 지정해주면 객체를 생성하면서 속성을 지정할 수 있다.
